## 📱 项目介绍

中小学科创产品定制/批量化微信小程序，支持C端（学生/家长）浏览、定制、学习，B端（教师/学校）批量采购，以及管理员后台管理。

## 🛠️ 技术栈

- **前端**：Taro 4 + React + TypeScript + TailwindCSS
- **后端**：NestJS + Drizzle ORM
- **数据库**：SQLite（开发）/ PostgreSQL（生产）
